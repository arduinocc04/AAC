You need libpng, ncurses, eigen.
`pacman -S libpng ncurses eigen`

Optionally, for documentation, You need doxygen.
`pacman -S doxygen`
`cd doc && doxygen doxy.conf`

# How to build
1. `mkdir build`
2. `cd build`
3. `cmake ..`
4. `make`
